Big con

Tracing is controlled on a per "component" basis, where a "component" is a name of the form aaa.bbb.ccc where aaa is the Most significant part; for example, the utilities library might be called "utils", the doubly-linked list "utils.dlist", and the code to destroy a list "utils.dlist.del"
